업체 검증 기준

부산달리기는 정보 제공 플랫폼입니다. 업체 정보를 게시하기 전 아래 항목을 점검하며, 점검 항목은 "검증"이 아닌 "기본 확인 절차"로 운영됩니다.

1. 사업자 정보 확인

  • 운영 주체(상호·연락처)가 명확히 확인 가능한가
  • 응대 채널(전화·메신저)이 안정적으로 운영되는가
  • 응대 시간이 일관되게 안내되는가

2. 가격 사전 고지

  • 코스·시간별 가격이 사전에 명시되어 있는가
  • 추가 비용(심야할증·외곽 출장비 등)이 사전에 안내되는가
  • 현장에서 추가 결제를 요구할 경우 사전 동의 절차가 있는가

3. 응대·예약 기준

  • 예약 확정·변경·취소 정책이 명확한가
  • 도착 예정 시간이 사전에 안내되는가
  • 응대 태도와 안내가 일관성 있게 유지되는가

4. 위생·관리 기준

  • 일회용 페이퍼류·일회용품을 사용하는가
  • 오일·도구 관리가 위생적으로 이루어지는가
  • 관리사 위생(손·복장)이 청결한가

5. 이용자 보호 기준

  • 이용자가 강도·자세 조절을 자유롭게 요청할 수 있는가
  • 중단·환불 요청에 대해 정해진 정책이 운영되는가
  • 분쟁 발생 시 응대 절차가 마련되어 있는가

검증 절차

  1. 업체 등록 신청 접수
  2. 사업자 정보 및 운영 채널 확인
  3. 가격·정책 자료 검토
  4. 실 이용자 후기 누적 확인
  5. 통과 시 노출 / 미통과 시 보완 요청 또는 등록 거절

등록 이후에도 후기·민원에 따라 재검토되며, 기준 미달 시 노출이 제한될 수 있습니다.
