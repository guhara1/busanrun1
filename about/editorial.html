A02 Editorial Policy

정보가 어떻게 만들어지는가.

Google의 "Who, How, Why" 가이드라인에 따라, 부산달리기의 모든 콘텐츠가 어떻게 만들어지고 검증되는지 공개합니다.

1. 정보 작성 원칙

  • 모든 가이드 콘텐츠는 부산달리기 편집팀이 직접 작성합니다. 외부 기고는 받지 않으며, 광고주가 직접 본문을 작성하는 일은 없습니다.
  • 지역·코스·가격 정보는 공개된 업체 자료 + 편집팀 모니터링 + 이용자 제보의 3개 채널을 교차 확인하여 작성됩니다.
  • AI 보조 도구를 사용하더라도, 최종 검수 책임은 명시된 편집자에게 있습니다.

2. 5단계 업체 검증

단계점검 항목처리
1단계사업자 정보 (운영 주체·연락처·응대 채널)미흡 시 보완 요청
2단계가격 사전 고지 (코스·시간·추가 비용 사유)미공개 시 게재 거절
3단계응대·예약 정책 (확정·변경·취소)모호 시 명확화 요청
4단계위생 관리 (일회용품·도구·관리사)기준 미달 시 게재 보류
5단계이용자 보호 (강도 조절·중단·환불)정책 미비 시 게재 거절

전체 항목은 업체 검증 기준 페이지에 상세히 정리되어 있습니다.

3. 정보 갱신 주기

  • 코스·가격 안내 — 분기 1회 시장 평균 재확인
  • 지역별 가이드 — 후기·민원 반영 시 즉시 갱신
  • 업체 검증 상태 — 신규 후기 누적 시 자동 재검토 큐 등록
  • 법령·정책 관련 안내 — 변경 시 14일 이내 갱신

4. 광고와 정보의 분리

부산달리기는 광고 노출과 정보 게재를 명확히 분리합니다. 업체가 노출 확대를 원하더라도, 검증 기준을 통과하지 못한 경우 게재되지 않습니다. 게재된 업체에 대해 광고비를 받는 경우, 해당 사실을 페이지 내에 표시할 의무를 갖습니다(현재 광고 게재 형태는 운영되지 않습니다).

5. 사용자 피드백 반영

  • 이용 후기를 통해 접수된 문제는 영업일 기준 3일 이내 확인 후 업체에 사실관계 확인 요청을 보냅니다.
  • 반복 민원이 누적된 업체는 노출이 제한되며, 사실관계가 확인된 경우 게재가 중단됩니다.
  • 정보 정정 요청은 고객센터를 통해 접수할 수 있습니다.

6. 신뢰 신호 표기

  • 모든 가이드 페이지 하단에 편집자명·최종 갱신일을 표기합니다.
  • 편집팀 구성은 편집팀 소개 페이지에서 공개합니다.
  • 구조화 데이터(Schema.org)를 통해 작성자·발행처·갱신 정보를 검색엔진에 제공합니다.
