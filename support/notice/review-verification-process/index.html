신규 후기 인증 절차 안내

이용 후기의 신뢰도 강화를 위해 후기 작성 시 인증 절차가 추가됩니다. 4월 5일 작성분부터 적용됩니다.

DATE 2026-04-05 BY 부산달리기 편집팀

이용 후기는 부산달리기의 가장 중요한 신뢰 신호 중 하나입니다. 광고성·과장성 후기를 줄이고 실제 이용자의 정확한 정보를 더 잘 노출하기 위해 후기 인증 절차를 개정합니다.

변경 사항

  • 예약 인증을 거친 이용자만 후기 작성 가능
  • 후기 작성 시 이용 코스·시간·지역 입력 (검수 자료로 활용, 개인 식별 정보는 비공개)
  • 광고성·과장성 표현 검수 강화
  • 동일 IP에서 단기간 다수 후기 작성 시 자동 보류 후 검수
  • 업체 측의 자체 작성·대필 후기 게재 시 즉시 삭제 및 게재 정지

적용 시점

2026년 4월 5일 작성분부터 적용됩니다.

기존 후기는 어떻게 되나요

기존에 게재된 후기는 그대로 유지되며, 변경 사항은 신규 후기에만 적용됩니다. 다만 이미 게재된 후기에 대해서도 광고성·허위 의심 신고가 접수되면 재검토 후 조치합니다.

후기 작성에 관한 안내

이용자 분들의 솔직한 후기는 다른 분들의 합리적인 선택을 돕습니다. 부정적인 후기도 사실에 기반한 것이라면 그대로 게재되며, 업체 측이 임의로 삭제 요청을 할 수 없습니다.

전체 후기는 실제 이용 후기 페이지에서 확인하실 수 있습니다. 후기에 대한 문의는 고객센터로 연락해 주세요.
